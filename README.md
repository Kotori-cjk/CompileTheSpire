# CompileTheSPire：故障机器人大冒险

## 项目简介

《CompileTheSpire》是一款基于 **C++ / Qt** 开发的元学习教育游戏。

项目面向北京大学程序设计实习课程的学习场景，将 C++ 中常见的“根据 `main` 函数调用方式反推类接口”的训练过程，设计成尖塔探索、线索收集、代码块筛选与最终组装判定的游戏流程。

玩家将扮演一名核心受损的故障机器人，在探索尖塔的过程中寻找修复自身的方法。每个房间都会提供一部分 `main` 函数线索或代码块奖励，玩家需要理解这些代码片段背后的 C++ 语义要求，并在有限背包容量下筛选出真正有用的类成员实现，最终完成类定义并通过编译祭坛的判定。

## 项目定位

本项目属于程序设计实习大作业中的 **元学习赛道**。

项目目标不是简单地做选择题或刷题工具，而是尝试将 C++ 面向对象程序设计中的接口推理过程游戏化，使玩家在交互过程中理解以下知识点：

- 构造函数
- 拷贝构造与对象初始化
- 引用返回与值返回
- 运算符重载
- 类型转换函数
- 类接口设计
- 局部正确与整体正确的区别

## 核心玩法

游戏采用“探索式代码构筑”的玩法。

基本流程如下：

```text
进入尖塔
→ 探索房间
→ 发现 main 函数线索
→ 收集代码块和管理背包容量
→ 击败小怪，获得阶段性奖励并解锁后续房间
→ 收集所需代码块后挑战Boss
→ 根据判定结果通关并进行结算
